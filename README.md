## 🚀 工作流

1. 需求 → Issue
2. Issue → 总指挥分析 → 设计文档
3. 设计文档 → 任务拆分 → 指派给 Claude Code / Codex CLI
4. 代码 → PR → 总指挥审查
5. 通过 → Merge → 汇报
