# NailGlam — Project Spec

> Dream Team 第一个项目：女士美甲灵感大全

## 产品定位
美甲灵感 Discovery 平台 — 一站式探索、学习、分享美甲设计

## 目标用户
- 18-35 岁女性
- 喜欢自己做美甲（DIY）
- 想找灵感的消费者
- 美甲爱好者/从业者

## 核心功能
1. **灵感画廊** — 瀑布流浏览美甲设计
2. **风格探索** — 7大风格分类筛选
3. **DIY教程** — 步骤化教学 + 视频
4. **创意社区** — 用户分享与互动

## 风格分类（7类）
1. 🇫🇷 法式优雅 — 经典/彩色/V型法式
2. 🎌 日系甜美 — 腮红甲/渐变/3D立体花
3. 🖤 欧美浮夸 — 水晶钻饰/金属铆钉/超长尖甲
4. 🌈 艺术创意 — 水彩晕染/大理石纹/几何
5. 🌿 自然清新 — 裸色/极简线条/磨砂
6. ✨ 节日限定 — 圣诞/万圣/情人节
7. 🔥 当季爆款 — 猫眼/镜面/磁石/温变

## 设计系统
- 色彩：玫瑰粉 #E8A0BF / 金色 #D4A574 / 暖白 #FDF8F7
- 字体：Inter（正文）+ Playfair Display（标题）
- 风格：欧美极简 + 轻奢感（Glossier × Pinterest）
- 动画：Framer Motion 微交互

## 技术选型
- Next.js 14 App Router
- TypeScript 严格模式
- Tailwind CSS
- Framer Motion
- 部署：Vercel（免费）
